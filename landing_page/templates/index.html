Stub

Stub is a simple C++ library which is meant to provide simple helper functions to ease sometimes tedious testing tasks.

Key features

Stub is especially helpful when testing template functions which takes hard-to-construct objects as parameters. With stub the task of making such tests short and self-contained, becomes much simpler.

  • Function calls: Remove boilerplate when stubbing function calls
  • Setting expectations: Makes it easy to set and verify expectations.

Usage

You can install Stub via git

Cloning

git clone https://github.com/steinwurf/stub.git!

Building

python waf configure
python waf build
